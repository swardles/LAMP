US County Information

This data is from the State Based Counties Gazetteer File, and can be found by following this link: https://www.census.gov/geo/maps-data/data/gazetteer2010.html. This data set holds information regarding the counties of all 50 states, Puerto Rico and Washington D.C. after the 2010 census and includes information such as US state abbreviations, GEO ID, institute code, name of county, population, housing units, land area (in both miles and meters), water area (in both miles and meters), latitude and longitude. Each line of the data is a specific county instance and holds all the data related to that county pertaining to the previously mentioned data fields (information). This site allows you, the user, to choose and display the data sorted by state, number of counties to be seen and population (both ascending and descending). This program is meant to give the user easy access to a few of the states' data that can be found in the large data set on US counties. Although the data is not test data, the current site is more of a test site, and does not allow the user to query for all data in the database.
